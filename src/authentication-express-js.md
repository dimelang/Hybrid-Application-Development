# Authentication

Authentication (autentikasi) adalah proses untuk memverifikasi identitas pengguna. Tujuannya adalah memastikan bahwa pengguna yang mengakses sistem benar-benar adalah user yang terdaftar.

Dalam aplikasi backend, autentikasi biasanya digunakan untuk:

- Login pengguna
- Mengamankan endpoint tertentu
- Menghubungkan request dengan identitas user
- Mengelola sesi atau token akses
